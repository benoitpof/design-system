# Excel template — POF v1.0

3 sheets ready-to-use, brand-aligned with v3.2.6 hard locks.

| Sheet | Purpose |
|-------|---------|
| `Dashboard KPI` | 6-card grid (3×2) showing top metrics |
| `Data` | 12-column tabular data with alternating row fills, frozen header |
| `Chart` | Bar chart with POF palette (navy + steel), no gridlines |

## Style applied

- Header rows : navy `#1C1F3B` fill, white text Poppins SemiBold 14pt
- Data rows : alternating `#FFFFFF` / `#F9FCFF`, no borders
- Numbers : right-aligned, thousand separator (`#,##0`)
- Money : `#,##0 €` format
- Percentages : `0%` format
- Caption rows : Raleway Italic 9pt `#6C757D`
- Chart series : `#1C1F3B` (primary), `#435D74` (secondary)

## How to use

1. Open `pof-dashboard-template.xlsx`
2. Replace data with your own
3. Add/remove KPI cards on Sheet 1 (each card = 3 merged rows × 4 columns)
4. Add rows on Sheet 2 (alternating fill auto-applied via the existing pattern)
5. Update chart data range on Sheet 3 if needed

## Notes

- Compatible Excel 365, Excel 2019+, Numbers (Mac), Google Sheets (with minor visual diff)
- Fonts fallback to Arial if Poppins/Raleway not installed
- v1.0 — basic. P&L 3-year template, balance sheet, cash flow to come in v1.1
