Atomiques · génération & log

S1 · ATOMIQUE À builder
DS-Dataviz-generator
Génère un asset visuel atomique : chart (16 templates), map (5 patterns), table. Mode quick (single-shot, QA auto) ou iterate (push Notion + boucle review Benoît). Cherche d'abord similar dans Notion Media Assets DB avant génération.

S4 · ATOMIQUE À builder
ds-feedback
Log brut des Rex et feedbacks dans la DB Notion DS Feedback. Appelée par DS-Dataviz-generator et ds-file-assembler en phase 7, ou directement par Benoît. Mode cluster pour pré-traitement avant ds-iterate hebdo.

Orchestrateur · assemblage de livrables

S2 · ORCHESTRATEUR À builder
ds-file-assembler
Skill orchestrateur. Assemble decks, reports A4, web pages, social posts depuis du contenu source (Notion ou prompt direct). Lit les règles depuis le repo, choisit les layouts, orchestre la création des assets via DS-Dataviz-generator (path A pour assets stratégiques, path B pour tactiques). Remplace l'ancien `ds-deploy`.
genere un deck build report position paper page web post LinkedIn ds-file-assembler

Capitalisation · learning loop

S3 · CAPITALISATION À builder
ds-iterate
Skill de capitalisation. Lit DS Feedback DB, cluster + diagnostic, propose des modifs sur memory/, rules/ ou layouts/ via PR GitHub. Mode manual ou scheduled (Monday 9h Paris). Niveaux de risque A/B/C/D : scheduled task se limite à A et B, C (tokens) et D (Skills/masters/DB) demandent validation formelle Benoît.
