Flux de génération (deck/report/web/social)

sequenceDiagram
    participant U as Benoît
    participant FA as ds-file-assembler
    participant DZ as DS-Dataviz-generator
    participant N as Notion
    participant G as GitHub SSOT
    participant O as Output

    U->>FA: brief (type=deck/report/web/social)
    FA->>G: lit docs/layouts/.md + rules
    FA->>N: cherche assets existants
    alt asset manque
        FA->>DZ: génère asset
        DZ->>G: lit rules + memory
        DZ->>N: cherche similar
        DZ->>N: pousse asset (état=En cours)
        DZ->>U: notifie pour review
        U->>N: commente
        DZ->>N: itère jusqu'à validation
    end
    FA->>O: assemble document final
    FA->>U: livre output
    U->>FBS: log Rex (optional)
    FBS->>N: stocke feedback
